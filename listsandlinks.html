Procedure to Create and View a Website on the Production Server

  1. Create file in gedit
  2. Run python on another tab in terminal to run localhost8000
  3. Go to localhost8000 and you should see all changes
  4. To view it on production server, type sftp username
  5. Then do "put filename"
  6. View it on the production server by typing url cmis-hsteacher/listsandlinks.html on localhost8000
